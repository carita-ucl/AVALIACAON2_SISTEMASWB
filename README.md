# FoodHere - Encontre os Melhores Pratos na Sua Cidade

### FoodHere é um aplicativo que permite aos usuários descobrir restaurantes na sua cidade com base no prato que eles desejam comer. Ao buscar por um prato específico, o app exibe uma lista de estabelecimentos que oferecem essa opção, tornando mais fácil para os usuários escolherem onde comer.

## Funcionalidades
Pesquisa por Prato: O usuário pode pesquisar por um prato específico, como "pizza", "sushi", "hambúrguer", entre outros.

Localização: O app exibe resultados com base na localização da cidade do usuario, garantindo que as opções sejam dentro de sua área.

Lista de Restaurantes: O usuário visualiza uma lista de restaurantes que oferecem o prato desejado.
